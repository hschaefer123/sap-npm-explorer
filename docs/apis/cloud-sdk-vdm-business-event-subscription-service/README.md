# @sap/cloud-sdk-vdm-business-event-subscription-service

This package contains the OData VDM for the [Business Event Subscription Service](https://api.sap.com/api/CA_BEH_SUBSCRIPTION_SRV) of SAP S/4HANA Cloud.
This service is part of the following communication scenarios: Business Event Handling Integration (SAP_COM_0121), OpenText Business Event Integration (SAP_COM_0007).
You can find additional documentation for this service on [help.sap.com](https://help.sap.com:00443/http.svc/ahp2/SAP_S4HANA_CLOUD/latest/EN/31/ca00d025e14973a1d9cd727cc09419/frameset.htm).
